Brands Kahani — Vite React + Tailwind prototype

Quick start:
1. npm install
2. npm run dev
(If you get PostCSS errors, run `npx tailwindcss init -p` and ensure postcss.config.cjs exists — it's included.)

This scaffold includes:
- React Router pages: Home, About, Services, Franchise, Contact
- Tailwind configured tokens in tailwind.config.cjs
- Framer Motion page transitions between routes
